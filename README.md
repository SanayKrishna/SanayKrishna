## Featured Projects

### [CloudScanner](https://github.com/SanayKrishna/CloudScanner)
> Event-driven cloud security framework for AWS S3

Continuously audits S3 buckets for misconfigurations, integrates CloudTrail + EventBridge for real-time threat detection, and automates remediation workflows. Alerts delivered via SNS and Slack. Built with **Python · FastAPI · Boto3 · Terraform**.

`AWS S3` `CloudTrail` `EventBridge` `Lambda` `SNS` `FastAPI` `Terraform`

---

### [AI-Powered SOAR Engine](https://github.com/SanayKrishna/Ai-Powered-Soar-Engine)
> ML-driven Security Orchestration, Automation & Response platform

Uses a `RandomForestClassifier` to score and prioritise security alerts, reducing analyst alert fatigue. Executes automated playbooks based on predicted severity and pushes real-time incident alerts to Slack. Interactive SOC dashboard built with Flask + Chart.js. **Live demo deployed on Vercel.**

`Python` `Flask` `Scikit-learn` `SQLite` `Slack API` `Chart.js`
